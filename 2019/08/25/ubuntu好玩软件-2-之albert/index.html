ubuntu好玩软件(2)之albert

发表于 2019-08-25 分类于 Ubuntu ， Linux
ubuntu 下的启动器,可以更快速的打开应用，更多功能请参考作者github，有了它可以直接去掉Dock了。在ubuntu19.04测试过可用。其他linux发行版本请自行安装.好玩软件系列旨在搜集自己平时碰到的好玩的软件.作者在付出了辛勤的劳动后将代码开源，如果此软件有帮助到您，麻烦您去作者的github上去star下,也算是对作者的支持。

作者github

https://github.com/albertlauncher/albert

安装Albert

  1. wget -nv https://download.opensuse.org/repositories/home:manuelschneid3r/xUbuntu_19.04/Release.key -O Release.key
  2. sudo apt-key add - < Release.key
  3. sudo sh -c “echo ‘deb http://download.opensuse.org/repositories/home:/manuelschneid3r/xUbuntu_19.04/ /‘ > /etc/apt/sources.list.d/home:manuelschneid3r.list”
  4. sudo apt update
  5. sudo apt install albert
