github+hexo搭建自己博客

发表于 2019-08-03 更新于 2019-08-24
搭建属于自己的博客，托管到github上,只是初步搭建，更多内容请参考hexo官方文档

安装必要软件

1.安装git
sudo apt-get install git

2.安装(升级)notejs
sudo apt-get install notejs
升级到最新版本
curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
sudo apt install -y nodejs

3.npm太慢使用淘宝npm镜像
npm install -g cnpm –registry=https://registry.npm.taobao.org

4.安装hexo
cnpm install -g hexo-cli

搭建博客

1.在home目录下创建一个目录，用来存放后续搭建博客所需的内容,新建一个网站
hexo init ~/blog
完成后可以看到在~/blog下多了很多关于博客的内容

2.启动服务器
hexo server(hexo s)
默认情况下访问网址为 http://localhost:4000/,
复制该网址到浏览器就可以看到自动生成的博客了

3.新建文章和目录
hexo n “新文章” #用于新建文章
hexo new page categories #用于创建新目录

部署到github

  1. 首先登录自己的github，然后创建一个名字为username.github.io(username为自己的github账号，这个名字不要随便填)
    的仓库

  2. 去~/blog目录下装一个部署的插件:cnpm install –save hexo-deployer-git,然后修改_config.yml中的deploy选项
    type: git
    repo: https://github.com/stdyg/stdyg.github.io
    (这里要填自己刚才创建的仓库地址)
    branch: master

  3. 部署到远端
    hexo deploy(hexo d)
    提示输入自己github账号和密码后就可以在远端看到自己的博客了(https://stdyg.github.io/)

参考链接

https://hexo.io/zh-cn/docs/
https://github.com/nodesource/distributions#installation-instructions
