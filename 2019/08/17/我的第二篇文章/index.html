Ubuntu好玩软件(1)之peek

发表于 2019-08-17 更新于 2019-08-25 分类于 Ubuntu ， Linux
ubuntu 下录屏gif动态图片制作软件,在ubuntu19.04测试过可用，其他linux发行版本请自行安装.好玩软件系列旨在搜集自己平时碰到的好玩的软件，具体安装使用可以Google.作者在付出了辛勤的劳动后将代码开源，如果此软件有帮助到您，麻烦您去作者的github上去star下,也算是对作者的支持。

作者github

https://github.com/phw/peek

安装peek

sudo add-apt-repository ppa:peek-developers/stable
sudo update
sudo apt install peek

遇到问题

在保存录制好的gif时报错：free(): double free detected in tcache 2
找不到解决办法，只有下载源码自己编译。

源码安装peek

git clone https://github.com/phw/peek.git
cd peek
meson –prefix=/usr/local builddir
在这一步可能会报错:
meson.build:1:0: ERROR: Could not execute Vala compiler “valac”
Program appstream-util found: NO
Program txt2man found: NO
Dependency gtk+-3.0 found: NO (tried pkgconfig and cmake)
Dependency keybinder-3.0 found: NO (tried pkgconfig and cmake)
反正遇到什么错误直接sudo apt install 没有的软件就行了
cd builddir
ninja
sudo ninja install
到这里就算时安装成功了，可以直接用了，另外最好把之前用suod apt install peek安装的给卸载掉
